# 04-03: JWT Validation Middleware and Role-based Authorization

## Objective
Add `Microsoft.AspNetCore.Authentication.JwtBearer` to both APIs, configure JWT validation against Keycloak, set up fallback authorization policy (deny by default), and add `AllowAnonymous()` to GET endpoints.

## Tasks

### Task 1: Add JwtBearer package
1. Add `Microsoft.AspNetCore.Authentication.JwtBearer` version to `Directory.Packages.props` (10.0.x matching runtime)
2. Add `PackageReference` to both `OpenCode.DragonBall.Api.csproj` and `OpenCode.Music.Api.csproj`
3. Verify: `dotnet build` passes

### Task 2: Create KeycloakRolesClaimsTransformation
Create `src/OpenCode.DragonBall.Api/Auth/KeycloakRolesClaimsTransformation.cs`:
- Implement `IClaimsTransformation`
- Parse `realm_access` claim JSON
- Extract `roles` array
- Add each role as `ClaimTypes.Role` claim
- Return transformed principal

Create `src/OpenCode.Music.Api/Auth/KeycloakRolesClaimsTransformation.cs`:
- Identical implementation
- Same logic, duplicated per API project (not shared in Domain due to ASP.NET dependency)

### Task 3: Configure JWT auth in Dragon Ball API Program.cs
Add to `src/OpenCode.DragonBall.Api/Program.cs`:
```csharp
using Microsoft.AspNetCore.Authentication.JwtBearer;
using Microsoft.IdentityModel.Tokens;

builder.Services.AddAuthentication(JwtBearerDefaults.AuthenticationScheme)
    .AddJwtBearer(options =>
    {
        options.Authority = "http://keycloak:8080/realms/OpenCode";
        options.Audience = "dragonball-api";
        options.RequireHttpsMetadata = false;
        options.TokenValidationParameters = new TokenValidationParameters
        {
            ValidateIssuer = true,
            ValidateAudience = true,
            ValidateLifetime = true,
            RoleClaimType = ClaimTypes.Role
        };
    });

builder.Services.AddAuthorization(options =>
{
    options.FallbackPolicy = new AuthorizationPolicyBuilder()
        .RequireAuthenticatedUser()
        .RequireRole("editor")
        .Build();
});

builder.Services.AddTransient<IClaimsTransformation, KeycloakRolesClaimsTransformation>();
```

Add middleware after `app.UseCorrelationId()`:
```csharp
app.UseAuthentication();
app.UseAuthorization();
```
